ANGULAR


¿Qué es Angular?

Es un framework para aplicaciones web desarrollado en TypeScript, de código abierto, mantenido por Google se utiliza para crear y mantener aplicaciones web de una sola página.

Objetivo

Su objetivo es aumentar las aplicaciones basadas en navegador con capacidad de Modelo Vista Controlador (MVC), en un esfuerzo para hacer que el desarrollo y las pruebas sean más fáciles.
Angular se basa en clases tipo "Componentes", cuyas propiedades son las usadas para hacer el binding de los datos. En dichas clases tenemos propiedades (variables) y métodos (funciones a llamar).

Características

Velocidad y redimiento

Generación de código.

Convierte plantillas en código optimizado para las máquinas virtuales de Javascript.

Universal

Ejecuta aplicaciones en distintos servidores para renderizarlo de manera instantanea obteniendo solo HTML y CSS.

División del código

El enrutador de componentes ofrece una división automática de códigos para que solo se cargue el codigo necesario para procesar una vista.

Productividad

Plantillas

Permite crear rápidamente vistas de interfaz de usuario con una sintaxis de plantilla simple y potente.

Angular CLI

La herramientas de línea de comandos permite empezar a desarrollar, añadir componentes, testear y previsualizar la aplicación.

IDEs

Obtén sugerencias de código inteligente, detección de errores y otros comentarios en los IDEs más populares.

Desarrollo

Testing

Utiliza Karma para realizar pruebas unitarias, y Protactor para pruebas end to end de forma rápida y eficaz.

Animación

Angular permite crear animaciones complejas y de alto rendimiento con muy poco código a través de su API.

Accesibilidad

Posee características para crear aplicaciones accesibles mediante los componentes disponibles para ARIA.

Ventajas

La arquitectura de Angular Js enlaza JavaScript y HTML, el código de ambos ya está sincronizado. Por lo tanto, el framework ahorra mucho tiempo a los desarrolladores

Estructura de código.

Angular brinda plantillas; lo que te permite producir aplicaciones con código limpio. No solo ahorra tiempo, sino también facilita la modificación o reparación de las aplicaciones

Pruebas.

El framework permite el uso de pruebas unitarias y de integración

Futuro brillante.

El futuro de Angular es brillante debido a su funcionalidad y popularidad. Su base de usuarios sigue creciendo y tiene una gran cantidad de documentación en profundidad que se actualiza constantemente.

Compatibilidad móvil y de escritorio.

Angular puede ejecutarse en la mayoría de los navegadores web. No solo en computadores de escritorio sino también en dispo sitivos móviles.
